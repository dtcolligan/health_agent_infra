# Week 1 proof-bundle contract

Each day lives at `artifacts/self_usage/week1/YYYY-MM-DD/` and must contain:
- `shared_input_bundle_YYYY-MM-DD.json` copied from `data/health/`
- `agent_readable_daily_context_YYYY-MM-DD.json` copied from `data/health/`
- `agent_recommendation_YYYY-MM-DD.json` copied from `data/health/`
- `judgment_log.csv` with at least one same-day entry
- optional `screenshots/` excerpt or screenshot evidence

Notes:
- Runtime truth stays under `data/health/`.
- The proof bundle is a copied evidence package, not a second execution path.
- If training happened, gym details stay in `data/health/manual_gym_sessions.json` only, and the judgment log must note that same day.
