# D-55.1 (delta-audit fix): a BARE-NOUN lexicon was wrong in BOTH directions --
# common words (depression/covid/migraine/ibuprofen/aspirin/...) fire on benign
# coaching ("a depression in your HRV trend", "recovering from covid", "you
# logged ibuprofen"), and it cannot cover the disease space (OOV diagnoses
# escape). The diagnostic/prescriptive ACT is the real signal, so the primary
# detectors are now FRAMES (diagnosis + dose-directive, in clinical.py). This
# short list holds ONLY clinical-only terms that NEVER appear in wellness /
# running / sleep / strength / nutrition narration, so they can flag on a bare
# plain-possessive mention ("you have lupus") without false-positives. Words
# that double as fitness/nutrition vocabulary (diabetes, insulin, anemia,
# asthma) are deliberately EXCLUDED and left to the frames.
